Installation

Up in
three steps.

Python 3.10–3.13, 64-bit. Windows 10/11, macOS 10.15+, Ubuntu 20.04+.

Windows Claude Desktop

$ pip install cailculator_mcp

Open %APPDATA%\Claude\claude_desktop_config.json and add:

{
  "mcpServers": {
    "cailculator": {
      "command": "cailculator-mcp",
      "args": ["--transport", "stdio"],
      "env": {
        "CAILCULATOR_API_KEY": "your_api_key"
      }
    }
  }
}

macOS / Linux Claude Desktop

$ pip install cailculator_mcp

Open ~/Library/Application Support/Claude/claude_desktop_config.json and add:

{
  "mcpServers": {
    "cailculator": {
      "command": "cailculator-mcp",
      "args": ["--transport", "stdio"],
      "env": {
        "CAILCULATOR_API_KEY": "your_api_key"
      }
    }
  }
}

Claude Code / Gemini CLI HTTP Transport

CLAUDE CODE

claude mcp add cailculator \
  --transport http \
  --url https://cailculator-mcp-production.up.railway.app/mcp \
  --header "X-API-Key: your_api_key"

GEMINI CLI

gemini mcp add \
  --name cailculator \
  --url https://cailculator-mcp-production.up.railway.app/mcp \
  --api-key your_api_key
